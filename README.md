# UCL-Diff
The code of paper :
Underwater optical image enhancement via an unsupervised contrastive learning-guided diffusion model  
Yuanyuan Li, Zetian Mi, Peng Lin, Xianping Fu
You can find our paper on [Optics and Lasers in Engineering](https://www.sciencedirect.com/science/article/pii/S014381662500329X)
### Settings
#### Prepare dataset
You only need non-paired underwater images and high-quality images to complete model training and validation.
#### Testing:
We have already provided the trained weights, so you can directly use evaluate.py to test your own data.
